SWING VIDEOGAME PROJECT

Adventure Game Developed in Roblox Studio using Lua.
Developed core gameplay, server infrastructure systems, including a Mob Pathfinding system and cross-server event synchronization using the MessagingService API. Managed player databases via the RobloxEngine API. Created 3D models in Blender and designed user interfaces with Adobe Photoshop and Illustrator.

Role

Lead Developer, UX/UI Designer, 3D Modeler

Goal

To design and develop an engaging adventure game in Roblox Studio, focusing on intuitive gameplay, immersive environments, and user-friendly interfaces. Every 3D component, and UI element done by hand.

Relevant Skills Used

Lua, Distributed Systems, Database Management, Game Design, UX/UI Design, 3D Modeling

Overview

In Swing, you enter a vibrant fantasy world where your objective is to acquire rare weapons, battle challenging mobs and other players, complete engaging NPC questlines, unlock powerful enchantments, and explore diverse biomes. Experience dynamic server-wide events as you strive to become the most powerful adventurer in the realm.

Personal Summary

As the Lead Developer and UX/UI Designer for Swing, I spearheaded the creation of an immersive adventure game within Roblox Studio. My responsibilities included developing core gameplay mechanics, including a Mob Pathfinding system and cross-server event synchronization using the MessagingService API. I also managed player databases through the RobloxEngine API, ensuring a seamless gaming experience. Additionally, I contributed to the game's visual appeal by creating 3D models in Blender and designing intuitive user interfaces with Adobe Photoshop and Illustrator. This project allowed me to hone my skills in game development, distributed systems, and user experience design, resulting in a captivating game that engages players in a dynamic fantasy world.
