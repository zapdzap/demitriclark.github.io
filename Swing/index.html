SWING VIDEOGAME PROJECT

Adventure Game Developed in Roblox Studio using Lua.
Developed core game mechanics and subsystems, Including a Mob Pathfinding system, Server Synchronization using the MessagingService API and deterministic randomness, and Player-Save persistence using the DataStoreService API. My creative work involved 3D modeling assets in Blender and designing functional UIs with Adobe Photoshop and Roblox Studio, as well as directing the look and feel of the game.

Role

Lead Developer, 3D Modeler, UX/UI Designer

Goal

To design and develop an engaging adventure game in Roblox Studio, focusing on intuitive gameplay, immersive environments, and user-friendly interfaces.

Relevant Skills Used

Lua, Distributed Systems, Database Management, Game Design, UX/UI Design, 3D Modeling

Overview

In Swing, you enter a vibrant fantasy world where your objective is to acquire rare weapons, battle challenging mobs and other players, complete engaging NPC questlines, unlock powerful enchantments, and explore dangerous regions. Experience dynamic server-wide events as you strive to become the most powerful adventurer in the realm.

Personal Summary

As the Main Developer and Designer for Swing, I lead the creation of an immersive adventure game within Roblox Studio. My responsibilities were developing core game mechanics and subsystems, and designing the look and feel of the game. Several of the notable things I developed include: a Mob Pathfinding system, Server Synchronization using the MessagingService API / deterministic randomness, and Player-Save persistence using the DataStoreService API. My creative work involved 3D modeling assets in Blender and designing functional UIs with Adobe Photoshop and Roblox Studio. This project allowed me to hone my skills in game development, distributed systems, and UI design, resulting in a captivating game that engages players in a dynamic fantasy world.
