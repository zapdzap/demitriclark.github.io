NOTELEY WEBAPP

Notetaking Webapp programmed in Javascript, PHP, and SQL
Developed a web-based note-taking and organization application using JavaScript and PHP, featuring a user-friendly interface designed with Spline. Managed user notes through a SQL database, ensuring efficient data storage and retrieval.

Role

Sole Developer, UX/UI Designer

Goal

To create a web-based note-taking application that allows users to efficiently organize and manage their notes through an intuitive interface and robust backend functionality.

Relevant Skills Used

JavaScript, PHP, SQL, UX/UI Design, Spline, Database Management

Overview

Noteley is a web-based note-taking application designed to help users efficiently organize and manage their notes. Developed using JavaScript and PHP, the application features a user-friendly interface created with Spline, allowing for seamless navigation and interaction. The backend is powered by a SQL database, ensuring robust data storage and retrieval capabilities. Users can create, edit, and categorize their notes with ease, making Noteley an essential tool for students, professionals, and anyone looking to keep their thoughts organized in a digital format.

Personal Summary

As the Sole Developer and UX/UI Designer for Noteley, I was responsible for the entire development lifecycle of the application. This included designing the user interface with Spline to ensure an intuitive user experience, coding the frontend functionality using JavaScript, and implementing the backend logic with PHP. I also managed the SQL database to handle user data efficiently. This project allowed me to hone my skills in full-stack web development, database management, and user-centered design, however, the project is currently in a paused state as I focus on other endeavors. I look forward to resuming development in the future to further enhance its features and usability.
