NUTRIAPP

Nutrition Tracking App developed in Java using Spring Framework
Developed a nutrition tracking application using Java and the Spring Framework, enabling users to log meals, monitor calorie intake, and track macronutrient distribution. Implemented RESTful APIs for seamless data interaction between the front-end and back-end.

Role

Developer

Goal

To create a comprehensive nutrition tracking application that allows users to log their meals, monitor calorie intake, and track macronutrient distribution through an intuitive interface and robust backend functionality.

Relevant Skills Used

Java, Spring Framework, RESTful APIs, JSON, CSV

Overview

As a Developer for NutriApp, I was responsible for designing and implementing the core functionalities of the nutrition tracking application. This included developing RESTful APIs using the Spring Framework to facilitate seamless communication between the front-end and back-end. I implemented the functionality for created user registered ingredients, recipes, and meals, which are key components to the application in terms of interacting with the rest of the system when calculating weight goals and macronutrient intake. This project enhanced my skills in Java programming, API development, and secure application design.

Personal Summary

This project is still ongoing! I will be adding more features such as a mobile app interface, and expanding the backend functionality to include more detailed nutritional analysis and personalized recommendations based on user data. Stay tuned for updates as I continue to develop and enhance NutriApp.
