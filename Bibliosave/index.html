BIBLIOSAVE

Library info system using Python & Flask RESTful API
Developed a comprehensive library information system using Python and Flask RESTful API to manage book inventories, user accounts, and borrowing records. Implemented secure user authentication and role-based access control to ensure data integrity. Designed an intuitive front-end interface with HTML, CSS, and JavaScript to enhance user experience. Integrated a PostgreSQL database for efficient data storage and retrieval, optimizing query performance for large datasets.

Role

Sole Developer, UI/UX Designer

Goal

To develop a robust library information system that streamlines the management of book inventories, user accounts, and borrowing records through a user-friendly interface and secure backend functionality.

Relevant Skills Used

Python, Flask, RESTful API, HTML, CSS, JavaScript, PostgreSQL, User Authentication, UI/UX Design

Overview

As the Sole Developer and UI/UX Designer for BiblioSave, I was responsible for the entire development lifecycle of the library information system. This included designing the user interface to ensure an intuitive user experience, coding the frontend functionality using HTML, CSS, and JavaScript, and implementing the backend logic with Python and Flask RESTful API. I also managed the PostgreSQL database to handle user data efficiently. This project allowed me to hone my skills in full-stack web development, database management, and user-centered design.

Personal Summary

This project is in the final stages of development, with core functionalities implemented and undergoing testing. I plan to add additional features such as advanced search capabilities, user reviews, and integration with external book databases to enhance the system's usability and functionality. Stay tuned for updates as I continue to refine and expand BiblioSave.
